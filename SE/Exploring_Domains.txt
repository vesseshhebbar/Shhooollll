Password Management Systems

  Features
    •	Master Password for the account
    •	2 Factor Authentication
    •	Encrypting all the passwords (Independent of master password)
    •	Password Sharing
    •	Structured data
    •	Searching
    •	Convenient Reminders to change passwords
    •	Suggesting High entropy passwords
    •	Browser plugin, browser integration
    •	Stored in the cloud
    •   Auto fill forms
    •	Web/Mobile/Desktop clients.
    •	Import/Export, Backup/Restore
    
  Use Cases
    •	Employees in an organisation
      o	Large Scale organisations
      o	Small Scale organisations
    •	Personal Use
    


   1. Create Account -
              Enter Personal Details.
              Obtain email-ID, phone number.
              Enter new password twice.
              Enter encryption key twice.
              Verify Captcha.
              Accept Privacy Policy and T&C.
              Verify with OTP.
      
      Login.
        - Challenge based authentication.
        - First login : Configure the account.
	      - Optional n step authentication
	      - Unlock passwords
      
    2. Configure the Account	
	     Enable/Disable n step auth.
	      Specify secure device
	      Appearance/Themes
	      Reminder Frequency
	      Default new password preferences

    3. Add Passwords
	      Add:
		      Title, website URL, username, etc.
	      Type password, or
	      Suggest Password with constraints: size/alphanumeric/spl char/unicode/entropy
	      Choose categories
	      Configure Owner, Sharing, Permissions, etc.
	      Add Optional Notes.

    4. Remind User
	      Implement notification queue
	      Auto-suggest password/copy to clipboard when notification is clicked.

    5. Shared Password
	      Share with other users/groups.

    6. Batch selection/modifications
	      Search
	      Actions

    7. Create Group/ Invite to group
	      Creator is admin
	      Create Invite Link.

    8. Create Category
	      Category Name
	      [Category Parent]
	      Sync category with these users/groups:
	      Force syncing on/off on all children

    9. Import Export Data
      	Into CSV, Keepass, etc.
